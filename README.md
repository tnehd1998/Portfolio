</br>


## 🧩 프로젝트 작동 화면

프로젝트의 사용 과정은 [작동 과정](https://github.com/tnehd1998/Portfolio/wiki)을 통해 확인하실 수 있습니다.

## 🛠 사용 기술

- **Library** : React, Styled-Components, Recoil
- **Deploy** : gh-pages

## 💻 사용 방법

```powershell
git clone https://github.com/tnehd1998/Portfolio.git
cd Portfolio
npm install
npm start

```
